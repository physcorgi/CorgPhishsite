Обзор

CorgPhish - это интеллектуальная система защиты от фишинга, разработанная для обнаружения и предотвращения фишинговых атак. Система использует алгоритмы машинного обучения для анализа ссылок и веб-страниц, выявляя подозрительные и потенциально опасные сайты.

CorgPhish предназначен как для индивидуальных пользователей, так и для организаций, желающих повысить свою кибербезопасность.

Установка

Существует несколько способов установки и использования CorgPhish:

Для разработчиков

Клонируйте репозиторий и установите зависимости:

git clone https://github.com/physcorgi/CorgPhish.git
cd CorgPhish
pip install -r requirements.txt

Для пользователей

Загрузите и установите приложение с нашей страницы загрузки.

Использование

После установки вы можете использовать CorgPhish следующими способами:

Браузерное расширение

После установки расширения в браузер, CorgPhish будет автоматически анализировать посещаемые вами сайты и предупреждать о подозрительных страницах.

Командная строка

Для проверки ссылки через командную строку:

corgphish check https://example.com

API

CorgPhish предоставляет RESTful API для интеграции с другими приложениями.

Примеры запросов

curl -X POST -H "Content-Type: application/json" -d '{"url": "https://example.com"}' https://api.corgphish.example.com/v1/check

Ответ API

{
  "url": "https://example.com",
  "is_phishing": false,
  "confidence": 0.98,
  "analysis": {
    "domain_age": "12 years",
    "ssl_valid": true,
    "suspicious_patterns": 0
  }
}

Часто задаваемые вопросы

Что такое фишинг?

Фишинг — это вид интернет-мошенничества, целью которого является получение доступа к конфиденциальным данным пользователей. Злоумышленники создают поддельные веб-сайты, имитирующие настоящие, чтобы украсть личные данные пользователей.

Как CorgPhish определяет фишинговые сайты?

CorgPhish использует комбинацию технологий машинного обучения, анализа URL, проверки SSL-сертификатов, анализа содержимого страницы и сравнения с базой данных известных фишинговых сайтов.

Совместим ли CorgPhish с моим браузером?

CorgPhish совместим с большинством популярных браузеров, включая Chrome, Firefox, Edge и Safari.

Как часто обновляется база данных фишинговых сайтов?

База данных обновляется несколько раз в день для обеспечения максимальной защиты от новейших угроз.
