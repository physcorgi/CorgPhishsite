Расширение для браузера

1. Скачайте установщик

Нажмите кнопку "Скачать для Windows" выше, чтобы получить последнюю версию.

2. Запустите установщик

Дважды кликните на скачанный .exe файл и следуйте инструкциям мастера установки.

3. Запустите CorgPhish

После установки вы можете найти CorgPhish в меню Пуск или на рабочем столе.

4. Интеграция с браузером

При первом запуске вам будет предложено установить расширения для Chrome, Firefox или Edge.

1. Скачайте DMG файл

Нажмите кнопку "Скачать для macOS" выше, чтобы получить последнюю версию.

2. Откройте DMG файл

Дважды кликните по скачанному .dmg файлу, чтобы смонтировать его.

3. Установите CorgPhish

Перетащите приложение CorgPhish в папку Applications.

4. Первый запуск

Кликните правой кнопкой мыши на приложении и выберите "Открыть", чтобы обойти предупреждения Gatekeeper при первом запуске.

sudo spctl --master-disable

Часто задаваемые вопросы

Является ли CorgPhish бесплатным?

Да, CorgPhish полностью бесплатен и имеет открытый исходный код. Вы можете использовать его для личных или коммерческих целей без каких-либо ограничений. Основной функционал всегда будет бесплатным, хотя в будущем мы можем ввести премиум-функции.

Как CorgPhish обнаруживает фишинговые попытки?

CorgPhish использует комбинацию алгоритмов машинного обучения, анализа URL, проверки содержимого и обнаружения визуального сходства для идентификации потенциальных фишинговых попыток. Наши модели постоянно обновляются на основе последних фишинговых техник и угроз.

Собирает ли CorgPhish мои личные данные?

CorgPhish уважает вашу приватность. По умолчанию он собирает только анонимную статистику использования для улучшения сервиса. Вы можете отказаться от этого сбора данных в настройках. Мы никогда не собираем личную информацию, такую как история просмотров, пароли или другие конфиденциальные данные.

Могу ли я использовать CorgPhish в своей компании?

Абсолютно! CorgPhish может быть развернут по всей вашей организации для защиты всех сотрудников. Для корпоративных развертываний с централизованным управлением, пожалуйста, свяжитесь с нами для получения дополнительной информации о наших корпоративных решениях.

Как я могу внести свой вклад в проект?

Как проект с открытым исходным кодом, мы приветствуем вклады! Вы можете внести свой вклад, отправляя отчеты об ошибках, запросы на функции или вклады в код через наш репозиторий GitHub. Ознакомьтесь с нашими руководствами по внесению вклада для получения дополнительной информации.
