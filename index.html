Инструкция по установке

1. Скачайте установщик

Нажмите кнопку "Скачать для Windows" выше, чтобы получить последнюю версию.

2. Запустите установщик

Запустите скачанный .exe файл с правами администратора.

3. Следуйте инструкциям установщика

Примите лицензионное соглашение и выберите директорию установки.

4. Завершите установку

После завершения установки запустите программу с рабочего стола или из меню Пуск.

1. Скачайте DMG файл

Нажмите кнопку "Скачать для macOS" выше, чтобы получить последнюю версию.

2. Откройте DMG файл

Дважды кликните по скачанному .dmg файлу, чтобы смонтировать его.

3. Установите CorgPhish

Перетащите приложение CorgPhish в папку Applications.

4. Первый запуск

Кликните правой кнопкой мыши на приложении и выберите "Открыть", чтобы обойти предупреждения Gatekeeper при первом запуске.

1. Скачайте пакет

Нажмите кнопку "Скачать для Linux" выше, чтобы получить последнюю версию .deb пакета.

2. Установка с помощью apt (Debian/Ubuntu)

sudo apt install ./corgphish-linux-1.2.0.deb

3. Или установка с помощью RPM (Fedora/RHEL)

sudo rpm -i corgphish-linux-1.2.0.rpm

4. Запуск из терминала

corgphish

Часто задаваемые вопросы

Является ли CorgPhish бесплатным?

Да, CorgPhish полностью бесплатен и имеет открытый исходный код. Вы можете использовать его для личных или коммерческих целей без каких-либо ограничений.

Как CorgPhish обнаруживает фишинговые попытки?

CorgPhish использует комбинацию алгоритмов машинного обучения, анализа URL, проверки содержимого и обнаружения визуального сходства для идентификации потенциальных фишинговых попыток.

Собирает ли CorgPhish мои личные данные?

CorgPhish уважает вашу приватность. По умолчанию он собирает только анонимную статистику использования для улучшения сервиса. Вы можете отказаться от этого сбора данных в настройках.

Могу ли я использовать CorgPhish в своей компании?

Абсолютно! CorgPhish может быть развернут по всей вашей организации для защиты всех сотрудников. Для корпоративных развертываний с централизованным управлением, пожалуйста, свяжитесь с нами.
